Logikkriger, Papirkriger Logic proof generator v0.1a

This is the very first version of the proof generator, so have patience, and be prepared for bugs.
Enter a logical expression to generate a truthtable and a semantic tableaux according to your propositional logical expression.

If you type any of the following keywords, it automaticly converts these to connectives:

  • NOT: ¬
  • AND: ∧
  • OR: ∨
  • RARROW: ⇒
  • RLARROW: ⇔
  • THUS: ⊨

Logikkriger has been tested with Chrome Version 20.0.1132.47, and Firefox 16.0.2.
